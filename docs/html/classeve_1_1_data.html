◆ getDataPointer()

virtual void* eve::Data::getDataPointer ( )
pure virtual

get pointer to a vector of type int, double or string (not for array data). Use this to retrieve a vector with all values Cast the pointer to a vector with type vector<int> or vector<double> or vector<string> depending on the return value ofgetDataType(). The size of the vector may be derived from getDimension()[0] This vector must be deleted after use.

Returns
ptr address of a vector<int> or vector<double> or vector<string> (may be NULL)
